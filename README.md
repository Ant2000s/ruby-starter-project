# Ruby Starter Project

![Unit Tests & Lint](https://github.com/Ant2000s/ruby-starter-project/workflows/Unit%20Tests%20&%20Lint/badge.svg)


## Install

```bash
gem install bundler
bundle install
```

## Run test

```bash
rspec
```

## Run rubocop

```bash
rubocop
rubocop -a # Auto-correct offenses
```
